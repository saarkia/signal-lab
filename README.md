# Signal Lab

Signal Lab is a Next.js customer journey simulation app built to test a full
Codex-to-GitHub-to-Vercel workflow.

It models a lifecycle orchestration surface with:

- scenario controls for retention, win-back, launch and service recovery use cases
- deterministic forecasting for conversion, revenue, fatigue and suppression
- a responsive journey canvas with staged audience movement
- channel allocation controls across push, email and in-app messaging
- live event telemetry and an interpretable decision model

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Validate

```bash
npm run lint
npm run build
```

## Stack

- Next.js App Router
- React 19
- Tailwind CSS 4
- lucide-react icons
